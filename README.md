# MUSIC_ARCHIVE
리액트를 사용해 만든 앨범 등록,조회 홈페이지

### 접속 사이트
https://dltnqls774.github.io/music_archive/

### 사이트 구성
- 구글 ID를 통한 로그인
- 앨범 등록 화면
- 앨범 조회 화면
- 앨범 클릭 시 상세조회 화면
- 장바구니 화면
